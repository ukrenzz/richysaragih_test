Test 01 - Programmer Syncore

1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework?

Framework adalah sebuah kerangka dalam pengembangan aplikasi. Framework dapat disusun dari satu bahasa pemrograman utama atau/dan dapat menggunakan berbagai jenis bahasa pemrograman. Adanya Framework pada suatu pemrograman dapat memudahkan developer dalam mengembangkan program atau aplikasi. Selain memudahkan, framework juga mempercepat kinerja developer. Hal ini karena sudah tersedianya susunan atau kerangka dalam framework tersebut. Tak jarang pula sebuah framework sudah dilengkapi dengan berbagai macam library ataupun tools yang dapat digunakan developer agar mempercepat pengerjaan.

2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?

  • If Else

    If Else (Percabangan) merupakan fungsi yang memberikan output dengan kondisi tertentu. Melalui permisalan, dapat dicontohkan seperti ini : 

    Jika harga kopi lebih besar dari 30.000, maka uang tidak cukup. Jika harga kopi kurang dari 30.000, maka lakukan pembelian!
    Maka dapat dipastikan, program akan melakukan pengecekan saldo dan melakukan perintah berdasarkan jumlah saldo dompet anda. Pada program (PHP) dapat dilakukan dengan perintah dasar seperti : 
      if($price > $balance){
    echo "Saldo tidak cukup";
  } else {
    // do transaction;
  }

    Pada percabangan, biasanya terdapat 3 kondisi, yaitu If, if else, else.

  • While

    While merupakan salah satu fungsi perulangan yang melakukan perulangan tanpa batas hingga menemukan kondisi tertentu. Fungsi ini biasanya digunakan jika ingin dilakukan perulangan yang tidak diketahui berapa banyak perulangan yang akan terjadi, namun diketahui kondisi agar perulangan dapat berhenti.

  • While Do

    Sama seperti While, konsep perulangan pada While Do hanya terdapat sedikit perbedaan. Fungsi ini akan menjalankan statement Do pada saat pertama sekali, lalu melakukan pengecekan terhadap kondisi di While. Jika pada running proses Do sudah menemukan kondisi yang sesuai, maka tidak akan dilakukan proses perulangan. Namun, jika kondisi belum sesuai, statement yang terdapat di fungsi Do akan diulangi terus menerus hingga sesuai kondisi yang dibutuhkan.

  • Switch Case

    Switch Case merupakan sebuah fungsi yang melakukan percabangan dengan pengecekan pada kondisi yang statis atau tetap. Fungsi ini biasanya digunakan jika kondisi yang ada tidak akan berubah. Layaknya Switch pada sebuah saklar lampu, ketika kondisi sesuai, maka akan menjalankan sebuah perintah dalam statement tersebut. Fungsi ini membutuhkan fungsi Break agar menghentikan statement.

  • For

    For merupakan sebuah fungsi perulangan yang umumnya digunakan untuk melakukan perulangan statement dengan jumlah perluangan sudah diketahui. Fungsi ini biasanya terdapat 3 atribut, yaitu Start (init counter), condition (test counter) dan Step (increment counter).

    Berbeda dengan While atau While Do, fungsi ini sulit untuk mendapatkan infinity loop karena jumlah perluangan dan step perulangan dapat diatur.

  • Foreach

    Foreach merupakan sebuah fungsi perulangan yang berfungsi untuk mengulangi items pada sebuah array.

3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure?

  • Constructor

    Constructor merupakan fungsi yang akan dijalankan pada saat sebuah Class dilakukan pemanggilan atau diinisialisasi. Pada fungsi ini, umumnya dilakukan inisialisasi dan definisi pada sebuah instance member.

  • Method

    Method merupakan fungsi-fungsi yang terdapat pada sebuah class. Fungsi ini dapat berupa static, private protected ataupun public. Method umumnya mengembalikan sebuah nilai (function), namun method ini juga dapat digunakan untuk menjalankan sebuah perintah tertentu tanpa harus mengembalikan nilai (procedure).

  • Class

    Class merupakan susunan object yang menciptakan suatu instant. Class dapat menampung kerangka dasar dan berisi berbagai method. Class pada umumnya dapat diartikan sebuah klasifikasi dengan berbagai sifat (method) didalamnya.

  • Object

    Object merupakan variable atau bagian dari suatu class yang tersusun dari kerangka class tersebut. Object merupakan entitas yang dapat diidentifikasi dan direpresentasikan.

  • Variable

    Variable sebuah penyimpanan data yang dapat berubah-ubah dengan tipe data yang tetap ataupun yang berbeda tergantung dari bahasa pemrograman yang digunakan. Variable dapat diubah secara langsung ataupun menggunakan sebuah fungsi tertentu.

  • Function

    Function adalah kumpulan statement yang dipetakan dalam sebuah himpunan dan dinyatakan dengan lambang atau nama. Function bertujuan untuk memudahkan pemanggilan coding yang berulang (reusable). Selain itu, function juga bertujuan untuk memudahkan pembacaan coding karena lebih rapi. Function akan mengembalikan nilai tertentu setelah menjalankan berbagai statement dengan tipe data tertentu.

  • Procedure

    Procedure mirip seperti Function. Namun, Procedure tidak mengembalikan nilai setelah menjalankan berbagai statement. Procedure umumnya digunakan untuk menjalankan perintah-perintah yang bersifat void.

4. Perbedaan method=”Post” dan “Get” ?

  • Post

    Post merupakan metode pengiriman data ke server dengan memberikan request dan data tersebut bersifat rahasia antara client dan server. Pada html, method post digunakan untuk mengirimkan data tanpa terlihat pada address bar. Sedangkan pada API, method ini digunakan untuk mengirimkan data dari client ke server agar data yang dikirim lebih aman.

  • Get

    Post merupakan metode pengiriman data ke server dengan memberikan request dan data tersebut bersifat tidak kredensial antara client dan server. Pada html, method post digunakan untuk mengirimkan data dan terlihat pada address bar. Method ini umumnya digunakan untuk mengambil data dari sebuah API.

5. Jelaskan konsep mengenai web service dan fungsi apa yang digunakan untuk memanggil web service client?

Web Service merupakan seperangkat protokol yang berfungsi untuk melakukan pertukaran data antara aplikasi dengan sistem yang berbeda. Web service bertujuan untuk pertukaran data dalam bahasa yang berbeda.

Bahasa XML digunakan untuk pemanggilan web service client. Selain menggunakan XML, dapat menggunakan berbagai library HTTP Request di setiap bahasa pemrograman. Contohnya adalah curl di bahasa pemrograman PHP.

6. Jelaskan mengenai MVC dan HMVC!

MVC (Model View Controller) merupakan pola arsitektur pada pengembangan aplikasi yang umumnya adalah Website. Adapaun Model berfungsi untuk mengelola bagian yang terhubung langsung dengan database. View berfungsi untuk mengelola tampilan yang dapat dilihat oleh pengguna. Sedangkan Controller merupakan penghubung antara model dan view melalui request dari pengguna.

Sedangkan HMVC merupakan konsep MVC yang hirarki atau pengguna MVC dalam setiap fitur. Jika pada konsep MVC, suatu aplikasi memiliki folder model, view dan controller untuk seluruh fitur. Berbeda dengan HMVC, setiap fitur memiliki arsitektur MVC dan memudahkan kompilasi serta lebih tertata rapi.

7. Sebutkan fungi curl() dalam php!

curl() merupakan library yang berfungsi untuk melakukan HTTP Request di bahasa pemrograman PHP. Client URL (cURL) akan berhubungan dengan server untuk mengirim request ataupun menerima response.

8. Bagaimana menampilkan sebuah element array dalam PHP ?

Array dapat ditampilkan dengan menggunakan fungsi perulangan for maupun foreach. Pada umumnya foreach digunakanuntuk menampilkan data dalam array karena lebih efisien.

9. Perbedaan HTTP dan HTTPS?

HTTP atau Hypertext Transfer Protocol merupakan sebuah protokol jaringan pada bagian application layer. Protokol ini berfungsi untuk proses transfer informasi, dokumen, gambar dan video antar komputer. HTTP menyediakan berbagai kumpulan perintah jaringan yang berfungsi untuk melakukan komunikasi antara web server dengan komputer client.

Sedangkan HTTPS merupakan protokol HTTP yang telah dikembangkan menjadi lebih aman. Protokol HTTPS telah ditambahkan protokol SSL yang berfungsi untuk mengenkripsi data pada saat melakukan proses request.

10. Jelaskan Apa itu API? sebutkan contohnya?

API atau Application Programming Interface merupakan interface yang dapat menghubungkan satu aplikasi dengan aplikasi lain. API berfungsi untuk mengirim permintaan dari client ke server (request) dan menerima data dari server (response).

API memiliki beberapa prinsip kerja, yaitu : API REST, API Websocket, API SOAP dan API RPC. API REST adalah yang paling populer dan sangat fleksible saat ini. API jenis ini melakukan input klient untuk menjalankan internal function dan mengembalikan hasilnya ke klien.

Beberapa contoh API yang umum misalnya API pokemon, BMKG, dan banyak lainnya.
